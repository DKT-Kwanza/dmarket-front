## 사용 방법
1. 저장소를 복제합니다: `https://github.com/DKT-Kwanza/dmarket-front.git`
2. 프로젝트 디렉토리로 이동합니다: `cd dmarket-front`
3. 필요한 의존성을 설치합니다: `npm install`
4. 개발 서버를 시작합니다: `npm start`
5. 브라우저에서 `http://localhost:3000`으로 접속하여 쇼핑몰을 확인합니다.
