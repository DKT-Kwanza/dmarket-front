## 👀 시작 가이드
### Requirements
- [Node.js 21.6.1](https://nodejs.org/ca/blog/release/v21.6.1/)
### Installation
``` bash
$ git clone https://github.com/DKT-Kwanza/dmarket-front.git
$ cd dmarket-front
$ npm install
$ npm start
```
